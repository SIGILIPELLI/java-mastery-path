# Java Mastery Path

A structured, module-wise Java training program that takes you from your first
`public static void main` to production-grade, master-level engineering — with
runnable code snippets in every module and a hands-on project at the end of
each level.

## How the program is organized

| Level | Focus | Modules |
|-------|-------|---------|
| [Level 1 · Entry](level-1/index.md) | Language foundations | 9 topics + 1 project |
| [Level 2 · Intermediate](level-2/index.md) | Idiomatic, testable Java | 9 topics + 1 project |
| [Level 3 · Advanced](level-3/index.md) | Concurrency, APIs, databases | 10 topics + 1 project |
| [Level 4 · Master](level-4/index.md) | Production systems & architecture | 9 topics + 1 capstone |

## How to use this site

- Work through each level in order — later modules assume earlier ones.
- Every topic page has runnable code snippets — copy them into a local `.java`
  file and compile/run with `javac`/`java`, or paste into your IDE.
- Each level ends with a project that combines everything learned in that level.
- Use the search bar (top of the page) to jump straight to a topic.
